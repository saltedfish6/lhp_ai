## shrink 收缩计算（收缩严格要算权重，但放大不严格）
- 溢出空间 300px

- 分摊的比例？
   flex-shirink:2:1    500:400 

    500- 300*500 需要分摊的比例
    400- 300*400 需要分摊的比例
- 分摊的权重
   总的权重？
   500*2+400*1=1400
   1000/1400= 0.714 71.4%
   400/1400 =0.286 28.6%

   500-300*0.714=168.6
   400-300*0.286=318.4

    300+33.33=333.33
