## http 请求
   -  请求行 GET http://www.baidu.com

      设置各种头部信息

        Content-Type: application/json
        Authorization: Bearer token 凭证
      }
   -  请求体
     GET 没有请求体
     POST 可以有请求体
     { role: "user", content: "你好" }
