# Hulk Chrome扩展

一个简单的Chrome扩展，可以将当前网页的背景颜色改为绿色。

## 功能

- 点击扩展图标打开弹出窗口
- 点击"改变背景颜色"按钮将当前网页的背景颜色改为绿色

## 安装方法

1. 下载或克隆此仓库到本地
2. 打开Chrome浏览器，进入扩展程序页面：`chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择包含此扩展的文件夹（即包含manifest.json的文件夹）

## 使用方法

1. 安装完成后，Chrome工具栏会出现Hulk扩展图标
2. 浏览任意网页时，点击Hulk图标
3. 在弹出的窗口中，点击"改变背景颜色"按钮
4. 当前网页的背景会立即变为绿色

## 文件结构

- `manifest.json`: 扩展的配置文件
- `popup.html`: 弹出窗口的HTML结构
- `popup.js`: 处理弹出窗口交互的JavaScript代码
- `icons/`: 包含扩展图标的文件夹
